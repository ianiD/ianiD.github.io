III. Operatii aritmetice si conditii

Capitolul II | Capitolul IV

Dupa cum s-ar putea sa stii, in JavaScript ai la indemana diverse operatii aritmetice.
Acestea sunt facute cu ajutorul operatorilor si operanzilor.
Luand ca exemplu: 2 + 2 = 4; + este operator; 2 si 2 sunt operanzi; 4 este rezultatul.
Operanzii pot fi variabile sau numere, booleeni, mesaje(siruri de caractere) etc.
Comparatie intre operatorii din mate si cei din JavaScript:
mate -> javascript
  +  ->  +
  -  ->  -
  x  ->  *
  :  ->  /
Al doilea lucru despre care vom discuta in acest capitol sunt operatiile conditionate. Acestea vor fii folositoare in multe cazuri.
Operatiile conditionate ne ajuta atunci cand trebuie sa executam un set de instructiuni daca o anumita conditie e respectata(adevarata) sau, in cazul in care conditia e falsa, sa executam alt set de instructiuni.
Acestea se rezolva cu autorul instructiunii if(in engleza: daca).
Sintaxa instructiunii if(cum se foloseste):
if(conditie){
      Instructiuni_adevarat
}
Daca e nevoie se poate adauga else:
if(conditie){
      Instructiuni_adevarat
}else{
      Instructiuni_fals
}
In cazul in care conditia este respectata/este adevarata, se vor executa Instructiuni_adevarat. Altfel, daca sunt specificate, se vor executa Instructiuni_fals.
Conditiile sunt formate tot din operatori si operanzi, insa de data asta operatorii sunt 'mai mare', 'mai mic sau egal', 'mai mic' etc.
Operatori:
mate -> javascript
  >  ->  >
  <  ->  <
  ≥  ->  >=
  ≤  ->  <=
  =  ->  == sau ===
  ≠  ->  != sau !==
Exemple de conditii:
2 + 2 >= 4     (ADEVARAT)
2 + 2 == 4     (ADEVARAT)
6 * 5 >= 3     (ADEVARAT)
6 + 9 < 5      (FALS)
